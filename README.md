# Business Landing Page (Next.js + Tailwind)

🚀 A modern and responsive business landing page built with Next.js 13 and Tailwind CSS.

## ✨ Features
- Hero section with call-to-action
- About section
- Services showcase
- Contact form with Formspree
- Mobile responsive
